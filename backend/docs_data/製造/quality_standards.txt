【製造業 品質管理基準・ISO規格（ISO9001）対応マニュアル】

■ 1. ISO 9001 品質マネジメントシステム (QMS) 7つの基本原則
1. 顧客重視（顧客ニーズの理解と顧客満足度の向上）
2. リーダーシップ（目的と方向性の統一、経営陣の関与）
3. 人々の積極的参加（全従業員の力量向上と品質へのコミットメント）
4. プロセスアプローチ（業務をプロセスとして管理し、効率と一貫性を確保）
5. 改善（継続的改善、PDCAサイクルの徹底）
6. 客観的事実に基づく意思決定（データ・情報・統計分析を用いた判断）
7. 関係性管理（サプライヤー等、利害関係者との相互利益の構築）

■ 2. 品質記録・検査記録の必須項目
後工程での追跡や監査に備え、以下の項目を電子データまたは所定の様式で確実に記録する。
- 製品名・部品番号（型番）
- ロット番号（製造単位）
- 製造日時・検査日時
- 担当者ID（作業者および検査者）
- 検査結果（合格 / 不合格 / 特採）
- 測定値・公差（設計上の許容範囲に対する実測値）
- 使用設備・治工具番号

■ 3. 不適合品（不良品）発生時の対応手順と是正処置
1. 隔離・識別（赤札の貼付等により良品と明確に区別し、流出を防止する）
2. 記録と報告（不良内容の写真撮影、数値データ、発生日時の記録）
3. 原因分析（4M分析および「なぜなぜ分析」による真因の特定）
   - Man（人）：作業ミス、教育不足
   - Machine（機械）：設備の劣化、治具の精度不良
   - Material（材料）：材料のバラツキ、不良部材の混入
   - Method（方法）：作業手順書の不備、無理な工程
4. 是正処置の立案・実施（暫定対策および恒久対策）
5. 効果確認（対策実施後の歩留まりや検査データによる有効性の検証）
6. 水平展開（他ライン・他製品への類似不良の未然防止）

■ 5. トレーサビリティ（追跡可能性）管理
サプライチェーン全体での品質保証のため、以下の工程順に一貫した追跡体制を構築する。
[ 原材料入荷 ] → [ 受入検査 ] → [ 工程内検査 ] → [ 完成品検査 ] → [ 出荷検査 ]
- 全工程で「ロット番号」を紐付けて記録し、不具合発生時の迅速なバックワード・トレース（原因遡及）およびフォワード・トレース（影響範囲の特定）を可能にする。

■ 6. 2026年向け 最新コンプライアンス対応事項
- カーボンニュートラル対応：製造工程（Scope1,2）およびサプライチェーン（Scope3）におけるCO2排出量算定と記録の義務化対応。
- サプライチェーン透明性確保：人権デューデリジェンス（人権DD法）に基づく、調達先のリスク評価と監査の実施。
- 電子帳簿保存法対応：品質検査記録、納品書、検査成績書のタイムスタンプ付与および電子保存の徹底。
